# simpleClock ： 简易的钟
---
## 项目介绍：  
实现一个简单的，按现实时间绘制的钟表类。  
以及可以播放声音的多媒体钟表类。  
---
## 开发环境：  
本项目c++语言实现，同时使用了VS2022+[Raylib](https://github.com/raysan5/raylib)图形库。
